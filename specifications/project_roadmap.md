# Project Roadmap & Acceptance Criteria

## Milestone 1: Infrastructure
- Setup Vite + Next.js + Tailwind + TypeScript.
- Connect Supabase for Google Auth.
- **Criteria:** User can log in. Basic page loads in < 1s.

## Milestone 2: Lobby & Player Management
- Create UI for up to 20+ players/teams.
- Mode and Song-Pack selection menu.
- **Criteria:** A session can be started with randomized player order.

## Milestone 3: Universal Player & Anti-Cheat
- Implement streaming widgets (Spotify/YouTube/Apple).
- Build the "Swipe-to-Unlock" solution protector.
- **Criteria:** Audio plays correctly; solutions remain hidden until Pilot action.

## Milestone 4: Game Modes Implementation
- Code the specific logic for all 5 modes.
- **Criteria:** Mode 3 successfully locks the answer input before music playback starts.

## Milestone 5: PNG Export & Printing Tools
- Create the card generator for meinspiel.de and DIY.
- **Criteria:** High-res PNG export with correct bleed margins (3mm).
